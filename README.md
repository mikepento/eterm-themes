## eterm-themes

This is a collection of themes for use with the [Eterm][old-eterm] terminal emulator. The themes were individually packaged and sourced as part of the Debian lenny and squeeze distributions, and the original packages can be downloaded here:

Debian Sources Archive: [eterm-themes 0.9.3.20060329-2][debian-packages]

I created this source archive to provide a single release package containing all of the themes for easier retrieval and inclusion in the [Arch Linux AUR (Arch User Repository)][aur], however, you may feel free to use release packages here for whatever purpose you see fit.

### Version

Note that I have changed the version from `0.9.3` to `0.9.6` to better match up my AUR releases with the Eterm package that I maintain.
